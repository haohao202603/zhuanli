# Patent Insight Tracker 产品需求文档（PRD）

- **版本**：v1.0
- **日期**：2026-04-10
- **适用范围**：多靶点项目（STING/KRAS/TYK2等）
- **目标用户**：药化科学家、IP分析师、项目负责人、管理层

## 1. 项目背景
当前专利监控大多停留在列表层，缺少对化学结构与SAR的持续追踪能力。为支持研发决策，需要构建“及时监测 + 深度解读 + 结构差异分析 + 报告输出”的一体化平台。

## 2. 产品目标
1. 及时发现新公开、新授权、法律状态变化。
2. 自动生成重点专利深度解读。
3. 识别母核并标记取代基改动，实现SAR趋势跟踪。
4. 自动输出面向不同受众的PDF报告。

## 3. 用户与场景
### 3.1 用户角色
- 药化科学家：关注母核演化、R位点变化与SAR启发。
- IP分析师：关注同族、法域、权利要求边界。
- 项目负责人：关注竞争格局、风险事件、研发建议。

### 3.2 核心场景
- 配置某靶点+公司白名单，定期接收新专利提醒。
- 查看某公司该靶点专利在近6个月的母核演化。
- 对比重点专利与历史专利，快速定位结构改动位点。
- 一键导出周报/月报/专题报告。

## 4. 功能范围
### 4.1 监控配置
- 靶点关键词与同义词（AND/OR/NOT）
- 公司白名单/黑名单
- IPC/CPC筛选
- 子结构/SMARTS筛选（可选）

### 4.2 数据采集与标准化
- 多源专利数据采集（元数据+文本+法律状态）
- 字段标准化、同族归并、去重
- 每日增量 + 周期全量回补

### 4.3 事件检测与告警
- 事件类型：新公开、新授权、法律状态变化、新法域进入
- 告警级别：P1/P2/P3
- 通知通道：邮件、企业IM、站内消息

### 4.4 深度解读
- 专利技术要点摘要
- 权利要求关键点提取
- 与历史专利差异说明
- 风险标签与优先级建议

### 4.5 SAR结构分析（重点）
- 化学结构抽取（文本优先，图像增强）
- 母核识别（Bemis-Murcko + 规则修正）
- R-group映射与标准位点编号
- 差异标注（新增/替换/删除）
- 时间趋势分析与可视化

### 4.6 报告导出
- 周报/月报/专题模板
- 结构图、差异图、趋势图内嵌
- 一键导出PDF

## 5. 非功能需求
- 时效：90%事件在24小时内触达
- 准确：同族去重准确率≥98%
- 可追溯：结论可回链到原文与结构来源
- 可扩展：支持多靶点模板化复制

## 6. MVP范围（首期）
1. 项目与监控规则配置
2. 数据采集、标准化、专利列表检索
3. 基础事件告警
4. 重点专利摘要
5. 母核识别 + 基础R位点差异标注
6. 周报PDF导出

## 7. 验收指标（KPI）
- 监控命中率≥90%
- 告警及时性（24h）≥90%
- 结构分析可用率≥85%
- 报告生成成功率≥98%

## 8. 里程碑建议
- Phase 1（4~6周）：监控、列表、告警、基础报告
- Phase 2（6~8周）：SAR核心分析与专利深读
- Phase 3（4~6周）：竞争格局与高级模板
